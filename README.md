# Chocolate Shipments Analytics Dashboard

This Power BI project provides a comprehensive analysis of global chocolate logistics, tracking shipment efficiency, revenue performance, and delivery reliability.The report utilizes a relational data model to transform raw shipping logs into actionable supply chain insights.

---

### 📊 Key Features

* **Executive Performance KPIs**: Real-time tracking of Total Revenue, Shipment Volume, and average transit times.
* **Logistics & Delivery Analytics**: Deep dive into "On-Time vs. Delayed" delivery percentages and carrier reliability metrics.
* **Geospatial Insights**: Interactive map visuals displaying global distribution routes and warehouse traffic.
* **Product Trend Analysis**: Month-over-month volume monitoring for specific product categories like Dark, Milk, and White chocolate.
* **Advanced Filtering**: Dynamic slicers allowing users to drill down by region, shipping status, and product type.

---

### 🛠️ Technical Stack

* **BI Tool**: Power BI Desktop (Version CY25SU12).
* **Data Modeling**: Relational schema involving `Orders`, `Shipments`, `Products`, and `Locations`.
* **DAX (Data Analysis Expressions)**: Custom measures for calculating time-intelligence metrics and delivery success rates.
* **Power Query**: Robust ETL processes for data cleaning and geographic data normalization.

---

### 📂 File Structure

* **`chocolate_shipments.pbix`**: The core Power BI project file containing the data model, DAX measures, and visual reports.
* **`Resources/`**: Contains background assets and theme JSON files for the dashboard UI.

### 🚀 Getting Started
1.  **Download**: Clone this repository or download the `.pbix` file.
2.  **Open**: Launch the file in **Power BI Desktop** (latest version recommended).
3.  **Explore**: Use the interactive tabs to navigate between the **Executive Summary**, **Logistics Detail**, and **Product Trends**.
